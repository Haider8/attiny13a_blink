# Attiny13A Blink

![Blink GIF](/blink.gif)

## Installing required tools and dependencies

```sudo apt-get install binutils gcc-avr avr-libc uisp avrdude flex byacc bison```

## Compilation

```make```

## Upload code to Attiny MCU

```make upload```
